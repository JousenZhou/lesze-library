### Usage
开发，构建、运行|体验

```bash
# 获取远程仓库代码
git clone https://github.com/JousenZhou/lesze-library
# 进入目录
cd lesze-library
# 安装依赖
yarn
# 启动项目
yarn styleguide
# 在浏览器访问 http://localhost:6060
```
